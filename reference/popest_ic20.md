# Intercensal population estimates, 2010-2020

Kansas City population estimates for 2010-2019. According to the [US
Census
Bureau](https://www.census.gov/programs-surveys/popest/technical-documentation/research/intercensal-estimates.html),
"Once produced, the intercensal estimates become the preferred series of
data for the decade."

## Usage

``` r
popest_ic20
```

## Format

- geographic_area:

  Area name

- est_base_apr_1_2010:

  April 1, 2010, resident population estimates base

- est\_\[year\]:

  Resident population estimate as of July 1 of the year

- census_apr_1_2020:

  April 1, 2020, census population count

## Source

[US Census Bureau, City and Town Intercensal Population Totals:
2010-2020](https://www.census.gov/data/tables/time-series/demo/popest/intercensal-2010-2020-cities.html)
